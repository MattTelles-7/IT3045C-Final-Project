# What to do with these files

## 1) Open the project
Open the folder in Visual Studio or VS Code.

## 2) Run it once
Use:
- dotnet restore
- dotnet ef database update
- dotnet run

## 3) Test the API
Use the Swagger page:
- http://localhost:5099/swagger

Click through each controller and show:
- GET
- POST
- PUT
- DELETE

## 4) Push to a public GitHub repo
Every teammate needs at least one commit.
Easy method:
- one person pushes the full starter project
- each other person makes one tiny change and commits it

## 5) Record the presentation
Keep it between 4 and 6 minutes.
Show:
- project overview
- database / models
- controllers
- Swagger / NSwag
- CRUD working
- who built what

## 6) Submit
Submit:
- public GitHub repo link
- presentation video
- anything else Canvas asks for
